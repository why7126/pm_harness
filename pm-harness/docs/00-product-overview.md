# 产品总览


## 产品目标

为瓷砖零售店店主提供统一的瓷砖信息查询入口，为企业内部员工提供瓷砖资料维护后台。

## 核心场景

- 店主查看瓷砖图片、规格、分类、基础信息。
- 企业员工新增、编辑、下架瓷砖产品。
- 企业员工维护分类、规格和产品图片。

## 核心能力

- 瓷砖信息管理
- 瓷砖图片管理
- 瓷砖视频管理
- 瓷砖分类与筛选
- 管理端维护
- Web与小程序展示
