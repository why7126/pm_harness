# S3 适配说明

> **[通用]** 默认保留结构；**[个性化]** 根据项目生成；**[条件启用]** 仅在使用 S3 或 S3 Compatible 时保留。

## 0. 文档定位 `[通用]`

本文定义 `{PRODUCT_NAME}` 使用 S3 或 S3 Compatible 对象存储时的 Bucket、Key、签名 URL、权限、生命周期和测试要求。

## 1. 初始化参数 `[个性化]`

| 参数 | 说明 | 示例 |
|---|---|---|
| `{S3_PROVIDER}` | 供应商 | AWS S3 / S3 Compatible |
| `{S3_ENDPOINT}` | Endpoint | 待确认 |
| `{S3_REGION}` | Region | 待确认 |
| `{S3_BUCKET}` | Bucket | `{PRODUCT_CODE}` |
| `{SIGNED_URL_POLICY}` | 签名 URL 策略 | 短期私有 URL |
| `{S3_TEST_COMMAND}` | 测试命令 | 待确认 |

## 2. 兼容重点 `[通用]`

- Bucket 默认私有，公开访问必须有业务理由。
- 签名 URL、分片上传、生命周期、元数据、加密和跨域必须实际验证。
- S3 Compatible 存储不得默认等同 AWS S3，高级能力需单独测试。

## 3. 初始化生成规则 `[通用]`

启用 S3 时保留本文；否则删除或标记为不适用。不得写入真实密钥、生产 Endpoint 或测试通过记录。
