## 1. 设计定位

**产品名称：** STONEX 石材数据库  
**目标用户：** 高端工程采购方、室内设计师、建筑师  
**核心定位：** 专业石材瓷砖信息平台，强调材质质感与数据专业性  
**设计关键词：** 工业感 · 克制 · 精密 · 高端

### 设计原则

- 深底色衬托材质本身，让产品纹理成为主角
- 金色仅用于价格、强调词、主 CTA 三处，严格克制
- 极细 `0.5px` 分割线代替块状背景切换，保持视觉连贯
- 所有圆角接近直角（`border-radius: 2px`），呼应工业切割感
- 签名元素：纹理色系横向速览条，以颜色作为导航维度

## 5. 组件规范

### 5.1 导航栏（Nav）

```
高度：52px
背景：#18160F
下边框：0.5px solid rgba(255,255,255,0.07)
内边距：0 28px
```

- **Logo：** 大写字母间距 `0.16em`，品牌字母用金色 `#C8A055` 标注
- **导航链接：** 默认 `rgba(237,232,223,0.45)`，激活态白色 + 底部 `1.5px` 金色下划线
- **右侧 CTA：** `询价 / 登录`，描边按钮，金色边框与文字

### 5.2 Hero 区

```
内边距：60px 28px 48px
布局：两栏 grid，1fr : 1fr
```

- 左栏：眉标 + 大标题 + 正文 + 按钮组 + 数据统计
- 右栏：3 块材质色块拼贴（比例 1:1 两行，左侧高块跨两行）
- 材质色块含底部 overlay 标注品名与规格

**按钮组：**

| 类型 | 样式 |
|------|------|
| 主按钮 | 金色实底 `#C8A055`，文字 `#18160F`，`padding: 11px 24px` |
| 幽灵按钮 | 透明底，白色半透明描边，图标 + 文字 |

**数据统计条：** 4 项横排，细线分隔，数字 `22px / 500`，标签 `11px / 0.4 alpha`

### 5.3 数据条（Data Strip）

```
布局：四等分网格
边框：上下各 0.5px 分割线，单元格右侧 0.5px 分割线
内边距：18px 28px
```

数字 `20px / 500`，标签 `11px` 弱色

### 5.4 搜索栏

```
背景：#211E16
外边框：0.5px solid rgba(255,255,255,0.1)
圆角：3px
高度：44px
```

三段结构：
1. **品类下拉**：右侧细线分隔，`11px`，弱色文字
2. **文本输入框**：占满剩余宽度，`13px`，placeholder 极弱色
3. **AI 找砖入口**：左侧细线分隔，金色文字 + 图标
4. **搜索按钮**：金色实底，贴右侧圆角

### 5.5 筛选标签（Filter Chips）

```
默认：透明底，0.5px 描边 rgba(255,255,255,0.12)，弱色文字
激活：rgba(200,160,85,0.12) 底，金色描边，金色文字
内边距：5px 14px
圆角：2px
字号：11px
```

### 5.6 侧边栏（Sidebar）

```
宽度：200px
内边距：24px 20px
右边框：0.5px 分割线
```

**区块标题：** `10px`，`letter-spacing: 0.12em`，弱色大写

**筛选项：**
- 复选框：`13px × 13px`，`0.5px` 描边，圆角 `2px`
- 激活态：金色填充 `#C8A055`
- 文字：`12px`，激活态白色，默认弱色

**价格区间滑块：**
- 轨道：`rgba(255,255,255,0.05)`，高度 `3px`
- 选中区间：金色 `#C8A055`
- 滑块圆点：金色，直径 `11px`

### 5.7 产品卡（Product Card）

```
背景：#211E16
边框：0.5px solid rgba(255,255,255,0.07)
圆角：3px
```

**结构：**
- 图片区：高度 `130px`，hover 时 `transform: scale(1.03)`，`transition: 0.4s`
- 信息区：`padding: 12px 14px 14px`
- 徽章行：flex 横排，间距 `5px`
- 产品名：`13px / 500`
- 规格说明：`11px`，弱色，`line-height: 1.5`
- 底部行：价格（左）+ 操作图标（右）

**价格：** `15px / 500`，单位 `/m²` 用 `10px` 弱色

**操作图标：** `28px × 28px`，`0.5px` 描边，`border-radius: 2px`，内含 Tabler outline 图标

### 5.8 精选卡（Featured Card）

```
跨列：grid-column: span 2
布局：左图右信息，1fr : 1fr
```

- 图片区：充满左侧，无固定高度（`min-height: 180px`）
- 信息区：`padding: 20px`，flex 列布局，上下撑满
- 规格网格：`2×2`，每格独立小卡片 `rgba(255,255,255,0.04)` 底色
- 底部：价格 + 收藏按钮 + 主 CTA

### 5.9 纹理速览条（Texture Row）

```
内边距：24px 28px
```

8 个色块横排 `flex: 1`，高度 `48px`，圆角 `2px`，底部 `9px` 颜色名称标注（绝对定位，偏移 `-18px`）

### 5.10 分页器

```
按钮尺寸：32px × 32px
圆角：2px
默认：0.5px 描边，弱色文字
激活：金色实底，深色文字
```

### 5.11 页脚（Footer）

```
背景：#100F0A
内边距：28px 28px 20px
上边框：0.5px 分割线
```

四列网格（`2fr 1fr 1fr 1fr`），品牌列含 logo + 简介，其余列含链接列表

底部栏：版权信息（左）+ 社交图标按钮（右）

## 9. 登录页（Web 管理端）

> **最高优先级视觉源**：`issues/requirements/REQ-0001-user-login/prototype/web/user-login.html`  
> Golden reference：`issues/requirements/REQ-0001-user-login/prototype/web/user-login.png`  
> 实现 CSS：`src/web/src/features/auth/styles/login-page.css`

### 实现策略（CSS Port）

- 登录页视觉 MUST 自 `user-login.html` port 为专用 stylesheet（`login-page.css`），React 仅负责 DOM 结构与 auth 交互。
- 颜色 MUST 引用 `globals.css` 的 `--color-*` token；TSX MUST NOT 含裸 Hex；材质 tile 渐变 MAY 使用 CSS 内 scoped variables。
- 登录页表单 MUST 使用原生 `<input>` / `<button>` + port CSS class；**MUST NOT** 依赖 shadcn `Input`/`Button`/`Checkbox` 默认皮相。
- 左栏 MUST 使用 CSS 材质拼贴（`material-board`），**MUST NOT** 以 JPG 全屏铺底替代。

### 布局

- 桌面（≥1024px）：左右 50% 分屏（`.login-shell` grid）；左栏 `.brand-panel` + 右栏 `.form-panel`
- 移动（<1024px）：隐藏左栏，表单居中，最大宽度 520px

### 品牌字体

- **STONEX Logo**：`var(--font-brand)`（Cormorant Garamond），letter-spacing `0.16em`，品牌金
- 正文与表单：系统 sans-serif

### 登录页色彩与间距

| 项目 | 值 |
|---|---|
| 右栏背景 | `--color-page` + radial glow（对齐 HTML `.form-panel`） |
| 表单 max-width | 420px（桌面）/ 520px（移动） |
| 输入框高度 | 64px（桌面）/ 52px（<640px） |
| 表单项间距 | 28px |
| 标题到表单 | 48px |
| 主按钮到第三方 | 56px |
| 主按钮高度 | 56px |

### 登录页组件态

| 控件 | 要求 |
|---|---|
| 输入框 | 透明底、`1px` emphasis 边框；focus 纯品牌金边框，无 ring-offset |
| 主按钮 | `.primary` 金色实底，深色文字，2px 圆角 |
| 复选框 | 18px `.box` + hidden native checkbox，金色选中态 |
| 企微入口 | `.wecom` 全宽 54px 横排 + `/icons/wecom.svg` |
| 语言切换 | 右上角 `.language` 边框按钮「简体中文⌄」 |

| 占位交互 | noop 或弱提示；**禁止**表单上方 notice 横幅 |
