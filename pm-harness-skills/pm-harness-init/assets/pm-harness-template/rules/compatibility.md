# 兼容性规范

- Web 端默认兼容主流现代浏览器；如需兼容旧浏览器，必须在 OpenSpec 中明确。
- 移动端、小程序、桌面端等平台遵守各自运行时限制。
- API 需考虑移动弱网、超时、分页、重试和幂等。
- 数据库、对象存储、部署系统的适配细节放在 `compatibility/`。
- 私有化、信创、国产数据库或多云适配必须补充兼容性矩阵和测试。
